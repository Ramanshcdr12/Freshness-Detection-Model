# 🍎 Freshness Detection Model

An AI-powered web application that analyzes fruits and vegetables from images and predicts their freshness percentage using Computer Vision and Deep Learning techniques.

🔗 **Live Demo:** https://freshness-detection-model-4qyp53hokxoxsjxj42xd6t.streamlit.app/

---

## 📌 Overview

Freshness Detection Model helps users determine the freshness of fruits and vegetables through image analysis. The application provides freshness scores, nutritional information, storage recommendations, visual explanations, and scan history in an intuitive Streamlit interface.

This project demonstrates the practical application of Deep Learning, Computer Vision, and Web Deployment in solving real-world food quality assessment problems.

---

## ✨ Key Features

### 🥬 Freshness Analysis

* Detects freshness percentage (0–100%)
* Identifies signs of spoilage
* Provides freshness-based recommendations

### 📊 Visual Insights

* Heatmap visualization of important image regions
* Confidence-based analysis
* Interactive result display

### 🍎 Nutrition Information

* Calories and nutritional values
* Vitamins and minerals
* Health benefits

### 🧊 Storage Guidance

* Shelf-life estimation
* Refrigeration recommendations
* Food preservation tips

### 📷 Multiple Input Methods

* Camera capture support
* Image upload support
* JPG, JPEG, and PNG formats

### 📜 Scan History

* Local scan history storage
* Review previous analyses
* Delete or clear saved records

### 🍛 Recipe Suggestions

* Indian recipe recommendations based on detected items

## 🧠 Model Architecture

The project utilizes **EfficientNetB0** with Transfer Learning:

1. Image preprocessing and normalization
2. Feature extraction using EfficientNetB0
3. Fruit/Vegetable classification
4. Freshness percentage prediction
5. Nutrition and storage recommendation generation
6. Heatmap-based visual explanation

### Training Strategy

* Pretrained ImageNet weights
* Transfer Learning
* Fine-Tuning of upper layers
* Regression-based freshness prediction

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Pranjal-1412/Freshness-Detection-Model.git
cd Freshness-Detection-Model
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```

Application will start at:

```text
http://localhost:8501
```
